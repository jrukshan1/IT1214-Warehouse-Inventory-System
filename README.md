# Warehouse Inventory Management System

## 📌 Project Description
This is a simple Java-based Warehouse Inventory Management System using Java Collections Framework (HashMap).  
It allows users to manage items in a warehouse through a console-based menu.

## ⚙️ Features
- Add new items
- Remove items
- Update item quantity
- Search items by ID or name
- Display all items
